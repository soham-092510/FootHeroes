# ⚽ FootHeroes — Football Match Centre & Tournament Manager

A full-featured football management app built as a single HTML file. No frameworks, no backend, no setup — just open in a browser.

Features:
- Role-based login — Tournament Head (admin) vs Fan/Player
- Live match clock with pause/reset
- Real-time scoreline with goal adjustment
- xG (Expected Goals) per player — updates live
- Win probability bar — recalculates on every event
- Score swing / momentum visualiser
- Time-based possession tracker (tap to set manually)
- Event logger — goals, cards, corners, subs, fouls with timestamps
- Penalty shootout module with kick cycling
- Visual football pitch with player dots (4-3-3 / 4-2-3-1)
- Player rating — auto-calculated from live stats
- Player stat editor (admin only)
- Knockout bracket generator (GC college format)
- Leaderboards — Golden Boot, Playmaker, Golden Glove, MVP
- 5 colour themes — Green, Cyan, Gold, Coral, default
- Community fan rooms and discussion cards
- Unique FHID generated per user

Tech: HTML · CSS · Vanilla JS — fully self-contained single file
